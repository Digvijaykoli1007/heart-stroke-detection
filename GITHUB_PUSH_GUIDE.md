## 🐛 Troubleshooting

### Error: "Repository not found"
➜ The repository doesn't exist on GitHub yet. Create it first (Step 1 above).

### Error: "Authentication failed"
➜ Generate a Personal Access Token or use GitHub CLI authentication.

### Error: "Updates were rejected"
➜ The remote has changes. If repository is empty, use: `git push -u origin main --force`
