  TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Problem: "DATABASE_URL is not set"
Solution: Update backend\.env with your Neon connection string

Problem: "Can't reach database server"
Solution: Neon databases auto-suspend after inactivity
         Visit https://console.neon.tech and click your project to wake it

Problem: Frontend shows "Network Error"
Solution: Make sure backend is running
         Run: cd backend && npm run dev

Problem: WebSocket shows "Disconnected"
Solution: Both frontend (port 3001) AND backend (port 5000) must be running

Problem: Port 5000 already in use
Solution: Change PORT=5000 to PORT=5001 in backend\.env
